## IMPLEMENTATION GUIDE

### How to Add These Meta Descriptions

**Option 1: React Component Method (Recommended for Single Page App)**

For a React SPA, you need to set meta tags dynamically. Use a custom hook or React Helmet:

```jsx
// In each page component
import { useEffect } from 'react';

export function CyberInsurancePage() {
  useEffect(() => {
    document.title = "Cyber Insurance for Businesses | Shoonya Insurance Brokers";
    const metaDescription = document.querySelector('meta[name="description"]');
    if (metaDescription) {
      metaDescription.setAttribute("content", 
        "Cyber insurance for businesses protecting against data breaches, ransomware, and cyber threats. Essential coverage for modern companies.");
    }
  }, []);
  
  return (
    // Page JSX
  );
}
```

**Option 2: HTML Method (If using server rendering)**

```html
<head>
  <title>Cyber Insurance for Businesses | Shoonya Insurance Brokers</title>
  <meta name="description" content="Cyber insurance for businesses...">
</head>
```

### Testing Your Meta Descriptions

1. **Google SERP Preview:**
   - Use Google's Mobile-Friendly Test
   - Check how title/description appear in search results

2. **Check Implementation:**
   - View page source (right-click → View Page Source)
   - Search for `<meta name="description"`
   - Verify content appears

3. **Tools:**
   - SEMrush Site Audit
   - Moz On-Page Grader
   - Screaming Frog Website Crawler
