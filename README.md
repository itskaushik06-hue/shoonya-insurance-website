# Shoonya Insurance Brokers - Premium Corporate Website

A premium, minimal corporate website for Shoonya Insurance Brokers, built with React and Tailwind CSS.

## Overview

This is a professional, trust-first website designed for a 75-year-old family-run insurance brokerage. The design emphasizes clarity, trust, and a human-centered approach to insurance services.

## Design Philosophy

- **Trust-first**: Professional, calm, and reassuring design
- **Minimal**: Clean white backgrounds with subtle grey accents
- **Human**: Focused on relationships, not transactions
- **Legacy-driven**: Emphasizes 75+ years of family tradition
- **Not salesy**: Editorial feel, no aggressive marketing

## Key Features

### Pages

1. **Homepage** 
   - Hero section with 75-year legacy messaging
   - What We Do (Insurance Audit, Tailor-Made Policies, Claim Support)
   - Insurance Categories (Individual, Business, Industrial, Specialized)
   - Trust & Legacy section
   - Client testimonials
   - Final CTA section

2. **Insurance Solutions Page**
   - Comprehensive listing of all insurance products
   - Organized by category (Individual, Business, Industrial, Specialized)
   - Policy cards with descriptions and "Learn More" links

3. **Policy Detail Pages** (Reusable Template)
   - Human-focused intro
   - 4-card information grid (Why You Need It, What It Covers, Why Shoonya, Peace of Mind)
   - CTA section
   - Examples: Cyber Insurance, Health Insurance, Motor Insurance

### Components

- **Header**: Sticky navigation with logo, menu, and CTA buttons
- **Footer**: Structured footer with company details, registration info, and contact
- **CTAButtons**: Reusable primary and secondary CTA components
- **MobileMenu**: Responsive hamburger menu for mobile devices
- **PolicyDetailPage**: Reusable template for all policy pages

### Design System

#### Colors
- **Primary**: `#1e3a8a` (Deep professional blue)
- **Secondary**: `#25D366` (WhatsApp green)
- **Neutral Greys**: `#fafafa` to `#171717`
- **Background**: `#ffffff` (Clean white)

#### Typography
- **Font**: DM Sans
- **Headings**: Bold (700), with negative letter-spacing
- **Body**: Normal (400), generous line-height (1.7)

#### Spacing
- Consistent spacing scale from 4px to 64px
- Generous whitespace between sections
- Max-width containers for optimal readability

#### Components
- Cards with subtle borders and hover effects
- Rounded corners (0.5rem)
- Smooth transitions on all interactive elements
- Icon containers with 10% opacity primary background

## Navigation

The website uses hash-based routing for simplicity:
- `#` - Homepage
- `#insurance-solutions` - Insurance Solutions page
- `#cyber-insurance` - Cyber Insurance detail page
- `#health-insurance` - Health Insurance detail page
- `#motor-insurance` - Motor Insurance detail page
